# sistema-estoque-csharp
Sistema console para gerenciar estoque de produtos, incluindo registro, movimentações e leitura de dados a partir de arquivo JSON. O objetivo é simular operações reais de estoque utilizando conceitos básicos de orientação a objetos.

Funcionalidades

Leitura de produtos via JSON.

Classes para produtos, estoque e movimentação.

Registro de entrada e saída de produtos.

Atualização automática da quantidade em estoque.

Menu interativo no console.
